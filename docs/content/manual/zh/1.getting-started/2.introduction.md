欢迎来到 **TockDocs**，一个基于 [Nuxt UI](https://ui.nuxt.com) 构建的全集成文档解决方案。

## 什么是 TockDocs？

TockDocs 是一个基于 [UI 文档模板](https://docs-template.nuxt.dev/) 的主题。视觉样式开箱即用，您的重点应该放在使用 Markdown 语法和 [Nuxt Content](https://content.nuxt.com) 提供的 [MDC](https://content.nuxt.com/docs/files/markdown#mdc-syntax) 编写内容。

## 核心功能

本主题包含诸多功能以提升您的文档管理体验：

- **基于 [Nuxt 4](https://nuxt.com)**：使用最新的 Nuxt 框架，获得最佳性能。
- **由 [Nuxt UI](https://ui.nuxt.com) 构建**：集成完整的 UI 组件套件。
- **[MDC 语法](https://content.nuxt.com/usage/markdown)** **通过 [Nuxt Content](https://content.nuxt.com)**：支持 Markdown 与组件集成，实现动态内容。
- **[Nuxt Studio](/docs/manual/zh/getting-started/studio)** **兼容**：以可视化方式编写和编辑内容，无需掌握 Markdown 知识！
- **自动生成侧边导航**：根据内容结构自动生成导航。
- **全文搜索**：内置搜索功能，轻松发现内容。
- **优化的排版**：精雕细琢的排版设计，提升可读性。
- **深色模式**：根据用户偏好支持深色模式。
- **丰富功能**：探索主题，发现全部能力。
