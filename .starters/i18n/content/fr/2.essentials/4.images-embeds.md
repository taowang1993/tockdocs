## Markdown

Affichez des images ou des vidéos en utilisant la syntaxe Markdown standard.

### Images

::code-preview
![Image Nuxt Social](https://nuxt.com/new-social.jpg)

#code

```mdc
![Image Nuxt Social](https://nuxt.com/new-social.jpg)
```

::

Ou avec vos images locales

::code-preview
![Montagnes enneigées dans une mer de nuages au coucher du soleil](/mountains.webp)

#code

```mdc
![Montagnes enneigées dans une mer de nuages au coucher du soleil](/mountains.webp)
```

::

::note{to="https://image.nuxt.com/"}
TockDocs utilisera le composant `<NuxtImg>` sous le capot à la place de la balise native `img`.
::

### Vidéos

Utilisez le composant `video` intégré avec un fichier média local ou distant.

```mdc
:video{autoplay controls loop src="/videos/demo.mp4"}
```
